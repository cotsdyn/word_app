# easelive-demo

A webapp that serves "Hello, {word from a database}" upon request.

## Running locally on a workstation

### Requirements
- Python 3.x
- python-pip
- python-venv

## Running in Docker

### Requirements
- Docker

## Information

Written in Python, using the [Flask](https://flask.palletsprojects.com/en/2.2.x/) web requests framework.
